# MIApp

MIApp é uma plataforma que permite a criação e execução de aplicativos desenvolvidos em PHP diretamente para desktop. Compatível com Linux (Debian e Ubuntu) e Windows.

A documentação formatada pode ser lida online em:
https://mestredainfo.wordpress.com/miapp/

## Mudanças no MIApp 3.0.0
A nova versão do MIApp será diferente da versão 2.0.0, pois agora permitirá a instalação de vários apps no mesmo executável. Essa decisão foi necessária para não ocupar tanto espaço em disco e facilitar o desenvolvimento para Linux e Windows.

Dessa forma usando o MIApp você pode desenvolver um único projeto em PHP para funcionar tanto no Linux quanto no Windows e reduz também o tamanho do arquivo para que os usuários possam baixar com mais rapidez, até mesmo se o usuário tiver uma conexão de Internet mais lenta, pois baixará apenas o projeto em PHP e não mais o executável do MIApp junto, para que o usuário possa executar os projetos em PHP, é necessário apenas ter um único MIApp.
